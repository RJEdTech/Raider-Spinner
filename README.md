# Raider Spinner

A random name picker and group maker built for Regis Jesuit High School teachers.

**[Launch Raider Spinner →](https://YOUR-USERNAME.github.io/raider-spinner/)**

## Features

- **Spinner** — Tap the wheel to randomly select a student. Toggle "Remove after picked" to avoid repeats (like pulling popsicle sticks).
- **Group Maker** — Instantly shuffle students into groups of 2, 3, 4, or 5.
- **Pick History** — See who's been called in the current session.

## Privacy

- No accounts, no login
- No ads, no cookies, no tracking
- No data leaves the browser — everything runs locally
- Nothing is stored when you close the tab

## How to Use

1. Visit the link above
2. Paste student names (one per line or comma-separated)
3. Click **Load Names**
4. Tap the wheel to spin, or switch to Group Maker
